# 17-OtherProvider

- 기본 트랙에 포함하지 않은 예제를 별도로 모은 폴더입니다.
- 무료 핵심 서비스 예제는 유지합니다.
- 일회성 유료 provider 예제는 코드 대신 제거 사유만 남깁니다.

## 포함

- LangSmith prompt hub
- Hugging Face endpoint / dataset upload
- LangSmith dataset / evaluation
- 멀티모달 전용 모델 예제

## 제거

- Google Gemini, Upstage, Pinecone, Cohere, Jina, LlamaParse
- 이유: 별도 유료 API 또는 특정 서비스 종속성이 강하고 현재 저장소의 기본 학습 흐름과 거리가 멉니다.
